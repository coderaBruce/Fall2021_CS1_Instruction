# Lab06 Instruction
## Mini-Sort (Call-by-Reference) (Milestone)
1. Reference is **alias** of another variable. Hence, change of reference does exactly the same of the referenced variable.
2. Declare a reference `<type>& <reference name>`. e.g., `double& a=b`

## Unique Numbers (C++ array)
https://www.cplusplus.com/doc/tutorial/arrays/ \
1. array is a fixed chunk of memories which store same type of things.
2. Declare array: `<type> <array-name>[]`
3. Initialize array: `int a[5]={1, 2, 3, 4, 5}; int a[10]={1, 2}; int a[10]={}`
